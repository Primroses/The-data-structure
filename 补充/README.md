# 时间复杂度
> 描述算法的性能和复杂程度
```
    O(1)        ---  常数
    O(log(n))   ---  对数
    O(log(n)c)  ---  对数多项式
    O(n)        ---  线性的
    O(n²)       ---  二次的
    O(nⁿ)       ---  多项式的
    O(cⁿ)       ---  指数
```

- 一、O(1)
```
function increment(num){
    return ++num;
}
```
假设运行 increament函数 执行时间是X，参数换了但是执行的时间还是X与参数无关，所以increment函数的性能都是一样的。所以这样的函数的时间复杂度是O(1);
